# **App Name**: AgroRegistro

## Core Features:

- Banana Wrapping Form: Form to register banana wrapping details: observation, color, quantity, and date.
- Cacao Harvest and Sales Form: Form to register cacao harvest and sales data: quantity, description, unit price, total value, and date.
- Banana Box Sales Form: Form to register banana box sales: quantity, tape color (linked to banana wrapping data), unit price, total price, and date.
- Dashboard Summary: Main dashboard summarizing the latest data from all forms.
- Data Storage: Storage of data from all forms

## Style Guidelines:

- Primary color: Earthy green (#8FBC8F) to evoke nature and agriculture.
- Background color: Light beige (#F5F5DC), a very slightly desaturated version of the primary.
- Accent color: Soft brown (#A0522D), to complement the green and beige, and offer good contrast.
- Font pairing: 'Alegreya' (serif) for headlines, and 'PT Sans' (sans-serif) for body text.
- Clean, intuitive form layouts with clear labeling and visual hierarchy.
- Use agricultural and data-related icons to represent data and navigation elements.
- Subtle transitions when navigating between forms and the dashboard.
